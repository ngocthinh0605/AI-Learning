# Mobile App (React Native) Scaffold

This directory contains a lightweight scaffold for the Sprint 6 mobile app track.

## Scope in this scaffold

- API client setup for the existing backend
- Auth and dashboard screen contracts
- IELTS and Rooms entry screen skeleton
- Basic Jest-style test files for API and screen logic

## Next implementation steps

1. Initialize full React Native or Expo runtime in this folder.
2. Wire navigation stack and authentication state persistence.
3. Reuse API contracts from `frontend/src/api/` where possible.
4. Add end-to-end tests once runtime is configured.
